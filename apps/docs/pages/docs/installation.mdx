# Installation

How to install dependencies and structure your app.

## Dependencies 

This UI is not framework specific and does not require you to install any dependencies except when using a component that uses another ui or library.

Pick the components you need. Copy and paste the code into your project and customize to your needs.

when a component uses another ui such as the shadcn's ui it will be indicated on the component's documentation and how to install the needed dependencies will be given.

**Some component ui and libraries used in  this ui are:**
- [shadcn ui](https://ui.shadcn.com/docs/installation)
- [Framer](https://www.framer.com/motion/introduction/##installation)


## TypeScript

This project and the components are written in TypeScript. We recommend using TypeScript for your project as well.
